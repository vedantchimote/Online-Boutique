## What is Online Boutique?

**Online Boutique** is a cloud-first microservices demo application. The application is a web-based e-commerce platform where users can browse items, add them to their cart, and purchase them.

Google uses this application to demonstrate how developers can modernize enterprise applications using Google Cloud products, including:

  </Card>
  <Card title="Cloud Service Mesh" icon="network-wired" href="https://cloud.google.com/service-mesh">
    Service mesh for secure service-to-service communication
  </Card>
  <Card title="Cloud Operations" icon="chart-line" href="https://cloud.google.com/products/operations">

## Key Features

<AccordionGroup>
  <Accordion title="Polyglot Microservices Architecture" icon="code">
    11 microservices written in 5 different languages (Go, C#, Node.js, Python, Java) demonstrating real-world polyglot architecture patterns.
  </Accordion>
  
  <Accordion title="gRPC Communication" icon="arrows-left-right">

  </Accordion>
  
  <Accordion title="Cloud-Native Design" icon="cloud">

  </Accordion>
  
  <Accordion title="Observability Ready" icon="eye">
    Integrated with Cloud Operations for monitoring, logging, tracing, and profiling out of the box.
  </Accordion>
  
  <Accordion title="Production Patterns" icon="shield-check">
    Demonstrates best practices including health checks, graceful shutdown, circuit breakers, and retry logic.

## Architecture Overview

Online Boutique consists of **11 microservices** that work together to provide a complete e-commerce experience:

| Service | Language | Purpose |
|---------|----------|---------|
| **Frontend** | Go | Web UI and API gateway |
| **Cart Service** | C# | Shopping cart management |
| **Product Catalog** | Go | Product inventory and search |
| **Currency Service** | Node.js | Multi-currency support |
| **Payment Service** | Node.js | Payment processing |
| **Shipping Service** | Go | Shipping cost calculation |
| **Email Service** | Python | Order confirmation emails |
| **Checkout Service** | Go | Order orchestration |
| **Recommendation** | Python | Product recommendations |
| **Ad Service** | Java | Contextual advertisements |
| **Load Generator** | Python | Traffic simulation |

## Why Online Boutique?

<Note>
  Online Boutique is designed to be **accessible to Kubernetes beginners** while demonstrating **production-ready patterns** for experienced developers.
</Note>

### For Learning
- Understand microservices architecture patterns
- Learn Kubernetes deployment and management
- Explore gRPC and Protocol Buffers
- Practice with cloud-native observability tools

### For Demonstrations
- Showcase Google Cloud Platform capabilities
- Demonstrate service mesh implementations
- Illustrate CI/CD pipelines
- Present monitoring and debugging workflows

### For Testing
- Benchmark Kubernetes performance
- Test observability tools and platforms
- Validate deployment automation
- Experiment with cloud services integration
